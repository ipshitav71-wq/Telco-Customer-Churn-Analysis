# 📊 Telco Customer Churn Analysis Dashboard

## 🔍 Project Overview
Interactive Power BI dashboard analyzing churn patterns 
across 7,043 telecom customers to identify key drivers 
and revenue at risk.

## 📁 Dataset
- Source: Telco Customer Churn Dataset (IBM Schema)
- Records: 7,043 customers
- Features: 28 columns including demographics, 
  services, and billing

## 💡 Key Insights
- Month-to-month contracts have 31.2% churn rate
  vs 12.8% for 2-year contracts
- 0-12 months tenure customers churn the most (35.8%)
- Electronic check users show highest churn (27.1%)
- $1.08M annual revenue at risk from churned customers

## 🛠️ Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions)
- CSV Data Processing

## 📊 DAX Measures Created
- Churn Rate
- Total Customers
- Churned Customers
- Revenue At Risk
